Como escrever um texto com parágrafos

(Voltar)

Ao criar sua estrutura de site em HTML, você pode utilizar de alguns recursos chamados "tags" que te auxiliam a construir uma estrutura estéticamente mais agradável. São diversas opções de tags, uma delas te ajuda a criar novos parágrafos em um texto.

Para criar um novo parágrafo, você precisa digitar a tag"< p >" antes de iniciar o mesmo. Ao finalizar o parágrafo, é só fechar o códico digitando "< /p >"

Como deixar o texto em negrito

(Voltar)

Se você deseja destacar uma palavra ou frase na sua página deixando-a em negrito, é só você incluir essa palavra ou texto desejado dentro da tag < strong > < /strong >

Como deixar o texto em itálico

(Voltar)

Se o que você precisa é deixar seu texto em itálico, você apenas precisa incluir o texto desejado dentro da tag < i > < /i >

Como incluir um campo de digitação

(Voltar)

Se no seu projeto você deseja incluir uma caixa para que o usuário digite alguma informação solicitada, você precisa incluir a tag < input > < /input >. Você também pode definir o tipo de informação que essa caixa vai receber, se for texto você coloca a tag < input type="text" >. Se o seu interesse for solicitar uma informação numérica ao usuário, você coloca a tag < input type="number" >. Observe abaixo um exemplo de caixa numérica
